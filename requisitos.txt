Desafio Frontend - Catálogo de
Produtos (React)
🎯 Objetivo do Desafio
Desenvolver uma aplicação web utilizando React que consuma a Fake Store API. O foco é
demonstrar habilidades em consumo de APIs, gerenciamento de estado, roteamento e
estilização.
🧩 Contexto do Sistema
Você criará um portal de compras onde será possível:
● Listar produtos por categorias.
● Visualizar detalhes de um produto específico.
● Simular a gestão de um carrinho de compras (CRUD local).
● Autenticar um usuário (Login).
🧩 Fake Store API
Para que este desafio foque inteiramente nas habilidades de Frontend, não será
necessário criar um banco de dados do zero. Utilizaremos a Fake Store API, uma API
pública e gratuita que simula um e-commerce real com produtos, categorias e usuários.
● Documentação Oficial: https://fakestoreapi.com/docs
Exemplos de Endpoints:
● Listagem de todos os produtos: /products
● Produto específico por ID: /products/:id
● Categorias disponíveis: /products/categories
🗓 Duração
● Duração total: 1 mês
● Divisão: 4 sprints (1 sprint por semana)
Cada sprint possui entregas obrigatórias.
🧱 Tecnologias
● React
● TypeScript (opcional, se aplicado)
● CSS / Styled Components / Tailwind (de acordo com a sua preferência)
● Fetch API para consumo de dados
